PointNet++: Deep Hierarchical Feature Learning on Point Sets in a Metric Space

1 Abstract

很少有先前的作品研究深度学习的点集。PointNet [20]是这方面的先驱。然而，通过设计，PointNet 不能捕获由所在的度量空间点引起的局部结构，从而限制了它识别细粒度模式和对复杂场景进行泛化的能力。在这项工作中，我们引入了一个递归应用 PointNet 的层次化神经网络，它在输入点集的一个嵌套划分上递归地应用 PointNet。通过利用度量空间距离，我们的网络能够学习与增加上下文尺度的局部特征。随着进一步观察到点集通常采用不同密度的样本，这导致在均匀密度下训练的网络性能大大降低，我们提出了新的集合学习层来自适应地组合来自多个尺度的特征。实验表明，我们的 PointNet + + 网络能够有效、稳定地学习深点集特性。特别是，在具有挑战性的3d 点云基准测试中，获得的结果明显优于最先进的技术。

2 网络结构

3 Hierarchical Point Set Feature Learning 层次点集特征学习

层次结构是由许多集合抽象层次组成的。在每个级别上，处理和抽象一组点，以生成一个元素较少的新集合。集合抽象层由3个关键层组成: 抽样层、分组层和 PointNet 层。采样层从输入点中选择一组点，这些点定义了局部区域的质心。然后，分组层通过寻找质心周围的“相邻”点来构造局部区域集。PointNet 层使用一个 mini-PointNet 将局部区域模式编码为特征向量。

  • Sampling layer

    • farthest point sampling (FPS)
      • 输入点云有N个点，从点云中选取一个点P0作为起始点，得到采样点集合S={P0}；
      • 计算所有点到P0的距离，构成N维数组L，从中选择最大值对应的点作为P1，更新采样点集合S={P0，P1}；
      • 计算所有点到P1的距离，对于每一个点Pi，其距离P1的距离如果小于L[i]，则更新L[i] = d(Pi, P1)，因此，数组L中存储的一直是每一个点到采样点集合S的最近距离；
      • 选取L中最大值对应的点作为P2，更新采样点集合S={P0，P1，P2}；
      • 重复2-4步，一直采样到N’个目标采样点为止。

4 Robust Feature Learning under Non-Uniform Sampling Density 非均匀采样密度下的鲁棒特征

点集在不同区域的密度不均匀是很常见的。这种不一致性给点集特征学习带来了巨大的挑战。在密集数据中学习的特征不能推广到稀疏抽样的区域。因此，针对稀疏点云训练的模型可能无法识别细粒度的局部结构。

理想情况下，我们希望尽可能近距离地观察一个点集，以便在采样密集的区域捕获最细微的细节。然而，这种近距离检查是禁止在低密度地区，因为局部模式可能被破坏的抽样不足。在这种情况下，我们应该在更大的范围内寻找更大的模式。为了实现这个目标，我们提出了密度自适应点网层(图3) ，当输入采样密度发生变化时，它们可以学习结合不同尺度区域的特征。我们称具有密度自适应 PointNet 层的分层网络为 PointNet + + 。

在前面的3.2节中，每个抽象层都包含单一尺度的分组和特征提取。在 PointNet + + 中，每个抽象层提取局部模式的多个尺度，并根据局部点密度智能地组合它们。根据对局部区域的分组和不同尺度特征的组合，我们提出了两种类型的密度适应层，

  • Multi-scale grouping (MSG)

    • 一个简单而有效的方法是使用不同尺度的分组层，然后根据“点网”提取每个尺度的特征。不同尺度的特征被连接起来形成一个多尺度特征。
    • 通过随机丢失每个实例的随机概率的输入点来实现的，我们称之为随机输入丢失。具体地说，对于每个训练点集，我们选择从[0，p ]中均匀采样的压降比 θ，其中 p ≤1。对于每个点，我们以 θ 的概率随机丢弃一个点。实际上，我们设置 p = 0.95是为了避免生成空点集。在此基础上，我们给出了具有不同稀疏度(由 θ 诱导)和不同均匀度(由丢失随机性诱导)的训练集的网络。在测试期间，我们保留所有可用的分数

  • Multi-resolution grouping (MRG)

    • 在图(b)中，某一水平上一个区域的特征 Li 是两个向量的连接。一个向量(图中左侧)是通过使用集合抽象层从 Li-1低层总结每个分区的特征得到的。另一个向量(右)是通过使用单个 PointNet 直接处理局部区域中的所有原始点而获得的特征。当局部区域密度较低时，第一个矢量的可靠性可能低于第二个矢量，因为在计算第一个矢量时，该分区包含更少的点，并且存在更多的抽样不足。在这种情况下，第二个向量的权重应该更高。
